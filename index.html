GiavaCms is a new java content management system.
You can create static or dynamic web pages using jee6 technologies cdi/ejb31 and jsf2.
/g:plusone
All web pages (static or jsf dynamic) are written to database and with private administration panel we can controll all things, without use ftp/ssh to publish new content). We use Prettyfaces (go to web site) to serve dynamically paths. The administration panel is developed using Primefaces (go to web site).
GiavaCms runs lovelly in redhat openshift (go to web site)
Initial modules: base (to create pages and templates, to controll users, to verify memory consumption, logs ), catalogue (simple category/product catalogue), contactus (register all contacts arrived from web form), gallery (to create photo gallery), message (to have comments in all place), richnews (rich news creator), scenario (it's product of catalogue aggregator), faq (Frequently Asked Questions).
Latest Modules: picasa, instagram, banner, exposition
Coming Soon Modules:paypal, blogger, facebook, twitter.
